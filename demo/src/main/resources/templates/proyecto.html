DESCRIPCIÓN DEL PROYECTO


El actual proyecto tiene como objetivo evaluar los conocimientos aprendidos en la clase de desarrollo web. De igual manera llevar a cabo un proyecto real que está relacionado con el mundo músical y que emplea varios de los conceptos aprendidos en la clase.

ENTREGA 1

Diccionario de datos

Tabla: UsuarioAdmon

Nombre de la columna Tipo de dato Descripción
ID Clave primaria Identificador único del usuario
Nombre Varchar El nombre del usuario administrador
Correo Varchar La dirección de correo electrónico del usuario administrador.
Password Varchar La contraseña del usuario administrador

Tabla: UsuarioVotante

Nombre de la columna Tipo de dato Descripción
ID Clave primaria Identificador único del usuario
Nombre Varchar El nombre del usuario votante
Correo Varchar La dirección de correo electrónico del usuario votante.
Password Varchar La contraseña del usuario votante

Tabla: Listas

Nombre de la columna Tipo de dato Descripción
ID Clave primaria Identificador único de la lista de canciones
genero Varchar El género musical asociado a la lista de canciones
UsuarioAdmonID Clave Foránea El ID del usuario administrador que creó la lista de canciones
UsuarioVontanteID Clave Foránea El ID del usuario votante que creó la lista de canciones

Tabla: Cnacion

Nombre de la columna Tipo de dato Descripción
ID Clave primaria Identificador único de la canción
nombre Varchar El nombre de la canción
genero Varchar El género musical al que pertenece la canción
rating INT El rating o calificación de la canción
artista Varchar El nombre del artista intérprete o grupo
album Varchar El nombre del álbum al que pertenece la canción
listasID Clave Foránea El ID de la lista de canciones a la que pertenece esta canción

Pruebas Unitarias

Clase Nombre de la prueba Descripció Resultados Esperados Resultados reales Cumplió
UsuarioAdmon Creación de Usuario administrador Verifica que se pueda crear un objeto correctamente. Los atributos deben coincidir con los datos proporcionados.
UsuarioAdmon Validación de correo único Comprueba que no se puedan crear múltiples usuarios administradores con el mismo correo. El segundo intento de creación debe fallar debido a la restricción de correo electrónico único..
UsuarioAdmon Actualización de contraseña Verifica que se pueda actualizar la contraseña de un usuario administrador correctamente. La contraseña después de la actualización debe coincidir con la nueva contraseña.
UsuarioAdmon Descativación de cuenta Confirma que se pueda desactivar la cuenta de un usuario administrador. El estado de activación de la cuenta debe cambiar a "inactivo" después de la desactivación.
UsuarioAdmon Eliminación de usuario Comprueba que se pueda eliminar un usuario administrador de la base de datos. El usuario administrador se debe eliminar correctamente de la base de datos.
UsuarioVotante Creación de Usuario votante Verifica que se pueda crear un objeto correctamente. Los atributos deben coincidir con los datos proporcionados.
UsuarioVotante Votación por canción Comprueba que un usuario votante pueda votar por una canción. El contador de votos de la canción debe aumentar en uno después de que el usuario votante haya votado.
UsuarioVotante Béusqueda; de cancién Confirma que un usuario votante pueda buscar una canción por nombre. Asegurarse de que la búsqueda devuelva la canción correcta según el nombre proporcionado.
UsuarioVotante Creación de lista de canciones Verifica que un usuario votante pueda crear una lista de canciones. La lista de canciones se haya creado correctamente y esté asociada al usuario votante.
UsuarioVotante Actualización de Datos del Usuario Votante Comprueba que un usuario votante pueda actualizar su información personal. Asegurarse de que los datos del usuario votante se hayan actualizado correctamente en la base de datos.
Listas Creación de lista de canciones Verifica que se pueda crear una instancia de Listas correctamente. Los atributos deben coincidir con los datos proporcionados.
Listas Actualización del nombre de la lista: Comprueba que se pueda actualizar el nombre de la lista de canciones. El nombre de la lista de canciones debe coincidir con el nuevo nombre especificado.
Listas Adición de Canción a la Lista Comprueba que se pueda agregar una canción a la lista. La lista de canciones debe contener la canción agregada.
Listas Eliminación de Canción de la Lista Verifica que se pueda eliminar una canción de la lista de canciones. La lista de canciones debe estar vacía después de eliminar la canción.
Listas Obtención de la lista de Canciones Confirma que se pueda obtener la lista de canciones asociada a Listas. La lista de canciones obtenida debe contener todas las canciones asociadas a Listas.
Cancion Creación de canciones Verifica que se pueda crear una instancia de 'Cancion' correctamente. Los atributos deben coincidir con los datos proporcionados.
Cancion Actualización del rating de la canción: Comprueba que se pueda actualizar el rating de una canción correctamente. El rating de la canción debe coincidir con el nuevo valor especificado.
Cancion Asociar una Canción a una Lista Verifica que se pueda asociar una canción a una lista de canciones correctamente. La canción debe estar asociada a la lista de canciones
Cancion Obtención de detalles de la Canción Confirma que se puedan obtener los detalles de una canción correctamente. Los detalles obtenidos deben coincidir con los datos de la canción creada.
Cancion Eliminación de Canciones Comprueba que se pueda eliminar una canción de la base de datos La canción debe eliminarse correctamente de la base de datos.
