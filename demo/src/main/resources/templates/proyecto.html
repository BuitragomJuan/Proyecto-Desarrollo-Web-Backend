DESCRIPCIÓN DEL PROYECTO


El actual proyecto tiene como objetivo evaluar los conocimientos aprendidos en la clase de desarrollo web. De igual manera llevar a cabo un proyecto real que está relacionado con el mundo músical y que emplea varios de los conceptos aprendidos en la clase.

ENTREGA 1

Diccionario de datos

Tabla: Usuarios - ‘ID’: Clave primaria, identificador único del usuario. - ‘Nombre de usuario’: Nombre de usuario único para autenticación. - ‘Contraseña’: Contraseña almacenada de forma segura. Tabla: Géneros Musicales - ‘ID’: Clave primaria, identificador único del género musical. - ‘Nombre del género’: Nombre del género musical. Tabla: Canciones - ‘ID’: Clave primaria, identificador único de la canción. - ‘Nombre de la canción’: Nombre de la canción. - ‘Artista’: Nombre del artista de la canción. - ‘Año de lanzamiento’: Año en que se lanzó la canción. - ‘Género musical’: Clave foránea que hace referencia a la tabla de géneros musicales. Tabla: Votos de Usuarios - ‘ID’: Clave primaria, identificador único del voto. - ‘ID del usuario’: Clave foránea que hace referencia a la tabla de usuarios. - ‘ID de la canción’: Clave foránea que hace referencia a la tabla de canciones. - ‘Valor de votación’: Valor numérico que representa la votación (positivo para voto, negativo para desvoto). Tabla: Listas de Reproducción por Género - ‘ID’: Clave primaria, identificador único de la lista de reproducción. - ‘ID del usuario’: Clave foránea que hace referencia a la tabla de usuarios. - ‘Género musical’: Clave foránea que hace referencia a la tabla de géneros musicales. Tabla: Registro de Actividad de Administradores - ‘ID’: Clave primaria, identificador único de la actividad. - ‘ID del usuario’: Clave foránea que hace referencia a la tabla de usuarios. - ‘Actividad realizada’: Tipo de actividad realizada por el administrador (creación, edición o eliminación de géneros o canciones). - ‘Fecha y hora de la actividad’: Registro de la fecha y hora en que se realizó la actividad.

Pruebas Unitarias

Clase Nombre de la prueba Descripció Resultados Esperados Resultados reales Cumplió
Usuario Creación de Usuario Verifica que un nuevo usuario pueda registrarse con éxito en la aplicación. Debe crearse un nuevo usuario en la base de datos y estar disponible para iniciar sesión.
Usuario Autenticación de Usuario Asegura que los usuarios puedan iniciar sesión de manera segura. Los usuarios autenticados pueden acceder a sus cuentas.
Usuario Actualización de Datos de Usuario Verifica que los usuarios puedan actualizar su información de perfil. Los datos del usuario se actualizan correctamente en la base de datos.
Usuario Eliminación de Usuario Comprueba que los usuarios puedan eliminar sus cuentas de usuario. La cuenta del usuario se elimina de la base de datos.
Usuario Validación de Contraseña Asegura que la aplicación valide y maneje adecuadamente las contraseñas de los usuarios. La aplicación debe rechazar contraseñas débiles o incorrectas.
Genero Creación de Género Musical (Usuario Administrador) Verifica que los usuarios administradores puedan crear nuevos géneros musicales. Debe crearse un nuevo género musical en la base de datos.
Genero Actualización de Género Musical (Usuario Administrador): Asegura que los usuarios administradores puedan editar la información de géneros musicales existentes. Los datos del género musical se actualizan correctamente en la base de datos.
Genero Eliminación de Género Musical (Usuario Administrador): Comprueba que los usuarios administradores puedan eliminar géneros musicales existentes. El género musical se elimina de la base de datos.
Genero Búsqueda de Géneros Musicales Verifica que la búsqueda de géneros musicales devuelva resultados precisos. Los resultados de búsqueda deben ser precisos y completos.
Genero Validación de Datos de Género Musical Asegura que la aplicación valide y maneje adecuadamente los datos de géneros musicales. La aplicación debe rechazar datos inválidos o incorrectos.
Canción Creación de Canción Verifica que se pueda agregar una nueva canción de manera exitosa. Debe crearse una nueva canción en la base de datos.
Canción Actualización de Canción: Asegura que se pueda editar la información de una canción existente. Los datos de la canción se actualizan correctamente en la base de datos.
Canción Eliminación de Canción Comprueba que se pueda eliminar una canción de la base de datos. La canción se elimina de la base de datos.
Canción Búsqueda de Canciones Verifica que la búsqueda de canciones devuelva resultados precisos. Los resultados de búsqueda deben ser precisos y completos.
Canción Votación de Canciones Asegura que los usuarios puedan votar y desvotar canciones correctamente, y que los votos se registren adecuadamente. Los votos se registran y actualizan correctamente en la base de datos.
